# Security Policy

## Supported Scope

OpenHamster is an auditable research and paper-trading system. Please report issues related to:
- authentication or unsafe API exposure
- secret leakage
- arbitrary code execution
- insecure file handling
- unsafe runtime override paths
- data integrity issues affecting audit records or paper ledger state

## Reporting

Please do not open public issues for suspected security problems.

Report privately to the repository owner through GitHub Security Advisories or a private maintainer contact channel if one is configured for the repository.

Include:
- affected version or commit
- reproduction steps
- impact assessment
- suggested mitigation if available

## Disclosure

- We prefer coordinated disclosure.
- After a fix is available, a public note can be added to release notes or repository security communication.
