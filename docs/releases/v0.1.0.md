# GobyShrimp v0.1.0

## Summary
`v0.1.0` 是 GobyShrimp 的第一版可发布基线。

这一版的重点不是“更多市场、更多数据、更多执行”，而是把一条可审计、可解释、可持续运行的研究与模拟盘主线做实：
- HK-only
- dynamic HK universe selection
- MiniMax live strategy research
- deterministic governance
- local paper ledger
- operator dashboard

## Highlights

### HK-only market-aware workflow
- 系统已切到 HK-only 主线
- universe 已升级为 `dynamic_hk`
- 当前研究标的不再固定死一个 symbol，而是从 HK 候选池中自动选出
- market profile 已进入 snapshot、prompt、governance 和 dashboard

### MiniMax live LLM path
- `LLM Gateway` 成为唯一 LLM 接入层
- MiniMax 为默认真实 provider
- prompt contracts 已模块化：
  - `market_analyst`
  - `strategy_agent`
  - `research_debate`
  - `risk_manager_llm`

### Deterministic governance
- proposal 不直接进入 paper
- 当前决策会显式输出：
  - `phase`
  - `next_step`
  - `resume_conditions`
  - ETA
  - blocked reasons
- governance 已包含：
  - promotion threshold
  - challenger delta
  - cooldown
  - pause / rollback semantics

### Local paper trading ledger
- active strategy 晋级后会写入本地 paper ledger
- 当前已支持：
  - orders
  - positions
  - daily NAV
  - continuous paper execution
- 页面会解释：
  - latest price freshness
  - whether price changed
  - whether equity changed
  - whether a rebalance was triggered
  - why no rebalance was required

### Runtime observability
- dashboard 已能看到持续运行状态：
  - `current_state`
  - `current_stage`
  - `last_run_at`
  - `last_success_at`
  - `consecutive_failures`
  - `expected_next_run_at`
- 支持：
  - manual sync
  - acceptance report
  - audit trail

### Macro-only context pipeline
- 当前事件输入已收敛为宏观单通道
- provider chain:
  - `FRED -> World Bank -> last known context`
- 系统会显式展示：
  - provider chain
  - reliability
  - freshness
  - degraded / fallback state

## Included Operator Views
- `/command`
- `/candidates`
- `/research`
- `/paper`
- `/audit`

## Validation Baseline
- `pytest tests -q` -> `122 passed, 8 skipped`
- `npm run build --prefix apps/web` -> passed

## Known Limits
- SQLite is still the default delivery database
- paper execution is simulated, not broker-connected
- long-horizon operating evidence still needs more runtime history
- macro context intentionally excludes news and announcements

## Upgrade Notes
- 旧的执行链路、通知链路、paper service 旁路和历史兼容层已被移除
- 项目叙事已统一为 GobyShrimp
- 当前发布重点是研究工厂与模拟盘治理，不是自动实盘
