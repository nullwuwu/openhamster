# 项目总览

## 新目标
构建一个由 LLM 与人类共同驱动的可审计策略工厂，而不是仅靠固定策略模板和参数调节运行的交易程序。

## 核心原则
- LLM 负责提出策略和解释市场
- `RiskManagerAgent` 拥有 B 级自治权，决定是否进入或留在模拟盘
- `ExecutionAgent` 必须是确定性执行层
- 审计链路必须可以回答“当时为什么允许这件事发生”

## 回测 / 模拟盘 / 实盘定义
- 回测：用历史数据离线验证策略，不产生真实订单
- 模拟盘：用在线或准实时数据运行策略，生成模拟订单与净值，但不动真实资金
- 实盘：通过券商接口使用真实资金下单

## B 级自治边界
- 自动创建研究任务
- 自动运行回测、walk-forward、研究辩论
- 满足门禁后自动进入模拟盘
- 不自动进入实盘

## 北极星指标
- 所有策略都有证据包与审计记录
- 模拟盘前必须有结构化放行结果
- 风险红线触发后必须可自动暂停或回滚
- 宏观日摘要已进入上下文和软评分
