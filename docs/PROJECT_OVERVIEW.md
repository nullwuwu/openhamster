## 一、项目定义

**项目名称**：港股模拟盘量化交易系统（HK Quant Sim）

**一句话描述**：基于 DualMA 策略 + Regime Filter，以自动化方式在港股模拟盘（2800.HK 盈富基金）上执行量化交易，并通过 IM（HoYowave）推送每日报告与风控告警。

**当前阶段**：P0 问题定义 / 架构对齐（尚未进入稳定开发态）

## 三、核心目标

1. **稳健跑赢基准**：年化收益接近恒生指数，MaxDD < 15%
2. **自动化运行**：每日自动执行策略信号判断、模拟交易、记账
3. **透明汇报**：每日收盘推送日报，异常时主动预警
4. **风控优先**：任何情况下不超过最大回撤红线

## 四、系统架构

```
用户（HoYowave IM）
    ↑ 日报 / 预警 / 熔断通知
OpenClaw Sub-agent（视察层）
    · 每日收盘后读取状态，生成日报
    · 回撤黄牌/熔断时 @用户
    ↓ 读取状态 & Metrics
State & Metrics（状态层）
    · 持仓记录 / 净值曲线 / 回撤计算
    · 存储：SQLite 或本地文件
    ↑ 写入
Trade Engine（执行内核）
    · 信号生成（DualMA + Regime Filter）
    · 模拟成交 / 记账
    · 硬风控（RiskGate：MaxDD≥15% 熔断）
    ↑ 唤醒
Scheduler / Daemon（调度层）
    · 每日港股收盘后（16:00 HKT）自动触发
    · 不依赖 OpenClaw 叫醒，trade-engine 旁边自己的 Daemon
    ↓ 拉取行情
数据源
    · 港股行情（待定：yfinance / Twelve Data / 其他）
    · 标的：2800.HK
```

## 六、风控设计

### 6.1 硬红线（RiskGate）

| 触发条件 | 动作 |
|---------|------|
| 当日回撤 ≥ 15% | 立即全部平仓（模拟），进入空仓状态，@用户熔断通知 |
| MaxDD（历史最大回撤）≥ 15% | 同上 |

### 6.2 黄牌预警

| 触发条件 | 动作 |
|---------|------|
| 当日回撤 ≥ 12% | 推送黄牌预警，@用户，不强制平仓 |

### 6.3 日常风控规则（来自 policy.yaml）

- 手续费：待确认（港股模拟盘标准费率）
- 杠杆：不允许（leverage = 1）
- 做空：不允许（allow_short = false）
- 最小数据年限：3 年（Walk-forward 验证要求）
- 参数敏感性黄线：> 50%（触发人工确认）

## 七、核心策略

### 7.1 DualMA（双均线交叉）

- **参数**：fast=50，slow=150（QQQ 验证最优参数，待 2800.HK 重新回测）
- **信号逻辑**：
  - 金叉（fast 上穿 slow）→ 买入
  - 死叉（fast 下穿 slow）→ 平仓

### 7.2 Regime Filter（市场状态门控）

- **指标**：MA 斜率 + ADX
- **逻辑**：
  - `trending` 状态：允许 DualMA 信号
  - `ranging` 状态：强制返回 0（不交易）
- **SPY 验证**：加入后 Test Sharpe +0.38（从负转正）

### 7.3 Walk-Forward 验证结论

- **训练集**：2022-2023 / **测试集**：2024-2025
- DualMA(50,150) QQQ：Test Sharpe 0.76，Robust ✅，PM 判决 GO
- ⚠️ 港股 2800.HK 尚未回测，需用港股数据重新验证

## 八、每日运行流程（SOP）

```
16:00 HKT 港股收盘
    ↓
Scheduler 触发 trade-engine
    ↓
拉取 2800.HK 当日收盘数据
    ↓
计算 DualMA 信号（经 Regime Filter）
    ↓
RiskGate 检查（MaxDD / 日回撤）
    ↓
模拟成交（买入 / 持有 / 平仓）
    ↓
更新 State & Metrics（持仓 / 净值 / 回撤）
    ↓
OpenClaw Sub-agent 读取状态
    ↓
生成日报推送至 HoYowave
    ├── 正常 → 日报（净值 / 持仓 / 今日信号 / 回撤）
    ├── 回撤 ≥12% → 黄牌预警 + @用户
    └── 回撤 ≥15% → 熔断通知 + @用户
```

## 十五、术语表

| 术语 | 含义 |
|------|------|
| DualMA | 双均线交叉策略（fast/slow MA 金叉买入、死叉平仓） |
| Regime Filter | 市场状态门控（MA斜率+ADX判断趋势/震荡，震荡时不交易） |
| Walk-forward | 滚动样本外验证（训练集选参数，测试集验证稳健性） |
| MaxDD | 最大回撤（历史净值从峰值到谷值的最大跌幅） |
| RiskGate | 硬规则风控门（触发红线直接熔断，不走LLM判断） |
| 熔断 | 回撤超红线时强制全部平仓、进入空仓状态 |
| 黄牌 | 回撤接近红线时的预警（不强制平仓） |
| 模拟盘 | 纸面交易，不动用真实资金，验证策略可行性 |
| 2800.HK | 盈富基金，港股恒生指数 ETF，流动性最好的港股 ETF |
| Sub-agent | OpenClaw 后台非阻塞子 Agent，负责读状态和汇报 |
